METEO 237 
    Une application de meteo permettant de consulter la meteo en temps reel d'une villle saisie par l'utilisateur,
    Elle offre une interface simple,rapide et intuitive pour afficher les informations météorologiques essentielles.
    
EXPLICATION DETAILLÉE ET AVANCEMENT DU PROJET
    L'application web de meteo est un mini projet sur lequel je travaille actuelement pour ameliore mes competences en developppemnt web 
    l'application en question fonctionne de sorte a ce que l'utilisateur entre le nom de la ville dont il veux connaitre la meteo et la page contenant les informations meteo s'affiche ; et l'utilisateur peux laisser son avis (concernant l'application web) dans un formulaire qui lui sera propose  ;
    Apres une semaine de travail j'ai deja eu a coder la partie frontend de mon application et dans les jours a venir je compte bien finaliser cela.
    En cas de changement ou d'ajout des fonctionnalités , cela sera notifié;
